üîµ Boolean Data Type

A boolean data type represents one of two values: True or False. The first letter T for True and F for False should be capital. Boolean values are used when we work with comparison operators.

Boolean Examples

print(True)
print(False)

# Boolean results from comparisons
print(3 > 2)     # True
print(3 < 2)     # False
print(3 == 3)    # True

‚öôÔ∏è Operators

Python language supports several types of operators. Operators are used to perform operations on variables and values.

Assignment Operators

Assignment operators are used to assign values to variables.

x = 10        # Simple assignment
x += 5        # Add and assign (x = x + 5)
x -= 3        # Subtract and assign (x = x - 3)
x *= 2        # Multiply and assign (x = x * 2)
x /= 2        # Divide and assign (x = x / 2)
x %= 3        # Modulus and assign (x = x % 3)

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations.

  • Addition (+): a + b
  • Subtraction (-): a - b
  • Multiplication (*): a * b
  • Division (/): a / b
  • Modulus (%): a % b (remainder)
  • Floor Division (//): a // b (division without remainder)
  • Exponentiation (**): a ** b (power)

Arithmetic Examples

print('Addition: ', 1 + 2)        # 3
print('Subtraction: ', 2 - 1)     # 1
print('Multiplication: ', 2 * 3)  # 6
print('Division: ', 6 / 2)        # 3.0
print('Floor Division: ', 7 // 2) # 3
print('Modulus: ', 3 % 2)         # 1
print('Exponentiation: ', 2 ** 3) # 8

# Real world example
radius = 10
area_of_circle = 3.14 * radius ** 2
print('Area of circle:', area_of_circle)  # 314.0

Comparison Operators

Comparison operators are used to compare values. They return either True or False.

print(3 > 2)      # True (greater than)
print(3 >= 2)     # True (greater than or equal)
print(3 < 2)      # False (less than)
print(2 < 3)      # True (less than)
print(2 <= 3)     # True (less than or equal)
print(3 == 2)     # False (equal)
print(3 != 2)     # True (not equal)

# String comparisons
print(len('mango') == len('avocado'))  # False (5 != 7)
print(len('mango') < len('avocado'))   # True (5 < 7)

Additional Comparison Operators

# 'is' operator - checks if variables are same object
print(1 is 1)  # True

# 'in' operator - checks if value exists in a sequence
print('A' in 'Asabeneh')  # True
print('B' in 'Asabeneh')  # False
print('coding' in 'coding for all')  # True

# 'not in' operator - opposite of 'in'
print('python' not in 'I love coding')  # True

Logical Operators

Logical operators are used to combine conditional statements. Python uses keywords: and, or, and not.

# 'and' operator - returns True if both statements are true
print(3 > 2 and 4 > 3)   # True
print(3 > 2 and 4 < 3)   # False

# 'or' operator - returns True if at least one statement is true
print(3 > 2 or 4 > 3)    # True
print(3 > 2 or 4 < 3)    # True
print(3 < 2 or 4 < 3)    # False

# 'not' operator - reverses the result
print(not 3 > 2)         # False (reverses True to False)
print(not False)         # True (reverses False to True)
