üì¶ Built-in Functions

Built-in functions are pre-defined functions in Python that are immediately available for use without importing anything. They perform common tasks and are essential for every Python programmer.

The print() Function

The most commonly used function - displays output to the console:

print("Hello, World!")
print(42)
print(3.14)
print("Multiple", "arguments", "work", "too")

The len() Function

Returns the number of items in an object (length):

name = "Python"
print(len(name))  # Output: 6

numbers = [1, 2, 3, 4, 5]
print(len(numbers))  # Output: 5

The type() Function

Returns the data type of an object:

print(type(42))           # 
print(type(3.14))         # 
print(type("Hello"))      # 
print(type([1, 2, 3]))    #

The input() Function

Gets input from the user via the keyboard:

name = input("What is your name? ")
print(f"Hello, {name}!")

The int(), float(), str() Functions

Convert values between different data types:

# String to integer
age_str = "25"
age = int(age_str)

# String to float
price_str = "19.99"
price = float(price_str)

# Number to string
number = 42
text = str(number)

The min(), max(), sum() Functions

numbers = [10, 5, 20, 15]

print(min(numbers))   # Output: 5
print(max(numbers))   # Output: 20
print(sum(numbers))   # Output: 50

üìù Variables

Variables are containers that store data values. You create a variable by assigning a value to a name using the equals sign (=).

Creating Variables

# Creating variables
age = 25
name = "Alice"
height = 5.6
is_student = True

# Using variables
print(name)
print(age + 5)
print(is_student)

Variable Naming Rules

  • Must start with a letter or underscore (_), not a number
  • Can contain letters, numbers, and underscores
  • Are case-sensitive (age, Age, AGE are different variables)
  • Should be descriptive and meaningful
  • Follow snake_case convention (use underscores for multiple words)

Good Variable Names

# Good - descriptive and clear
student_age = 20
first_name = "Bob"
is_active = True
user_emails = []

# Bad - unclear or invalid
a = 20
fn = "Bob"
x = True
123name = "Invalid"  # Starts with number

Multiple Variable Assignment

# Assign multiple variables at once
x, y, z = 10, 20, 30
print(x, y, z)  # Output: 10 20 30

# Swap values easily
a = 5
b = 10
a, b = b, a
print(a, b)  # Output: 10 5

üîÑ Type Casting

Converting values from one data type to another using built-in functions.

Converting to Integer

# From string
age = int("25")
print(age)  # 25

# From float (removes decimal)
whole = int(3.99)
print(whole)  # 3

Converting to Float

# From string
price = float("19.99")
print(price)  # 19.99

# From integer
decimal = float(42)
print(decimal)  # 42.0

Converting to String

# From integer
num_str = str(42)
print(num_str)  # "42"

# Useful for combining values
age = 25
message = "I am " + str(age) + " years old"
print(message)  # I am 25 years old

üî¢ Numbers in Python

Integers

Whole numbers without decimal points:

age = 25
temperature = -10
count = 0

Floats

Numbers with decimal points:

height = 5.9
pi = 3.14159
price = 19.99

Arithmetic Operations

x = 10
y = 3

print(x + y)   # Addition: 13
print(x - y)   # Subtraction: 7
print(x * y)   # Multiplication: 30
print(x / y)   # Division: 3.333...
print(x // y)  # Floor division: 3
print(x % y)   # Modulus: 1
print(x ** y)  # Exponentiation: 1000
