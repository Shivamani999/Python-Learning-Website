üìã Lists

A list is a collection of different data types which is ordered and modifiable (mutable). A list can be empty or it may have different data type items. There are four collection data types in Python: List, Tuple, Set, and Dictionary.

How to Create a List

In Python we can create lists in two ways:

# Using list built-in function
empty_list = list()
print(len(empty_list))  # 0

# Using square brackets
empty_list = []
print(len(empty_list))  # 0

# Lists with initial values
fruits = ['banana', 'orange', 'mango', 'lemon']
print('Fruits:', fruits)
print('Number of fruits:', len(fruits))

# Lists can have items of different data types
mixed_list = ['Asabeneh', 250, True, {'country':'Finland', 'city':'Helsinki'}]
print(mixed_list)

Accessing List Items Using Indexing

We access each item in a list using their index. A list index starts from 0.

# Positive Indexing
fruits = ['banana', 'orange', 'mango', 'lemon']
first_fruit = fruits[0]   # banana
second_fruit = fruits[1]  # orange
last_fruit = fruits[3]    # lemon

# Negative Indexing
last_fruit = fruits[-1]      # lemon
second_last = fruits[-2]     # mango
first_fruit = fruits[-4]     # banana

Slicing Lists

We can specify a range of indexes by specifying where to start and end the slice. The return value will be a new list.

fruits = ['banana', 'orange', 'mango', 'lemon']

# Positive slicing
all_fruits = fruits[0:4]  # all items
orange_and_mango = fruits[1:3]  # does not include index 3
orange_onwards = fruits[1:]  # from index 1 to end

# Negative slicing
orange_and_mango = fruits[-3:-1]  # from -3 to -1
reverse_fruits = fruits[::-1]  # reverse the list

Modifying Lists

Lists are mutable, so we can add, remove, or modify items.

# Modifying items
fruits = ['banana', 'orange', 'mango', 'lemon']
fruits[0] = 'avocado'
print(fruits)  # ['avocado', 'orange', 'mango', 'lemon']

# Adding items with append()
fruits.append('apple')
print(fruits)  # ['avocado', 'orange', 'mango', 'lemon', 'apple']

# Inserting items with insert()
fruits.insert(2, 'lime')
print(fruits)  # ['avocado', 'orange', 'lime', 'mango', 'lemon', 'apple']

# Removing items
fruits.remove('lime')  # remove specific item
fruits.pop()  # remove last item
del fruits[0]  # remove by index

Joining Lists

# Using + operator
list1 = [1, 2, 3]
list2 = [4, 5, 6]
combined = list1 + list2  # [1, 2, 3, 4, 5, 6]

# Using extend() method
fruits = ['banana', 'orange']
vegetables = ['tomato', 'potato']
fruits.extend(vegetables)
print(fruits)  # ['banana', 'orange', 'tomato', 'potato']

List Methods

# count() - returns occurrences of an item
fruits = ['banana', 'orange', 'mango', 'orange']
print(fruits.count('orange'))  # 2

# index() - returns index of first occurrence
print(fruits.index('mango'))  # 2

# reverse() - reverses the list in place
fruits.reverse()
print(fruits)

# sort() - sorts in ascending order
ages = [22, 19, 24, 25, 26, 24, 25, 24]
ages.sort()
print(ages)  # [19, 22, 24, 24, 24, 25, 25, 26]

# sorted() - returns sorted list without modifying original
ages = [22, 19, 24, 25]
sorted_ages = sorted(ages)
print(sorted_ages)  # [19, 22, 24, 25]
