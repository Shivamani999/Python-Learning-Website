üìò Introduction

Python is a high-level programming language for general-purpose programming. It is an open source, interpreted, object-oriented programming language. Python was created by a Dutch programmer, Guido van Rossum. The name Python was derived from a British sketch comedy series, Monty Python's Flying Circus. The first version was released on February 20, 1991.

This 30 days of Python challenge will help you learn Python 3 step by step. The topics are broken down into 30 days, where each day contains several topics with easy-to-understand explanations, real-world examples, hands-on exercises, and projects.

This challenge is designed for beginners and professionals who want to learn the Python programming language. It may take 30 to 100 days to complete, depending on your pace and dedication.

‚öôÔ∏è Environment Setup

Installing Python

To run Python code, you need to install Python. Visit python.org to download the latest version. Make sure you download Python 3.6 or above.

To verify Python is installed, run this command in your terminal:

python --version
# or
python3 --version

You should see output like: Python 3.x.x

The Python Interactive Shell

Python is an interpreted language that executes code line by line. You can test code immediately using the Python Interactive Shell. Open your terminal and type:

python

The shell is ready when you see the >> prompt. Now you can type Python code directly! For example:

>>> print("Hello, Python!")
Hello, Python!
>>> 2 + 3
5
>>> exit()  # Type this to exit the shell

Installing Visual Studio Code

For larger projects, use a code editor. Visual Studio Code is a popular, free choice. Download it and install the Python extension for the best experience.

üí° Tip: The Python interactive shell is great for quick tests, but use a code editor for actual projects.

üî§ Basic Python Syntax

Python Indentation

Python uses indentation (whitespace) to create blocks of code. Unlike other languages that use curly brackets, Python requires proper indentation. This is a common source of errors when learning Python!

if 5 > 3:
    print("Five is greater than three")  # This line must be indented

Comments in Python

Comments make code more readable and are ignored by Python during execution. Use # for single-line comments and triple quotes for multi-line comments:

# This is a single line comment

"""This is a multi-line comment.
It can span multiple lines.
Python ignores this block.
"""

üî¢ Basic Mathematical Operations

Python can perform various mathematical operations:

# Addition
print(2 + 3)      # Output: 5

# Subtraction
print(3 - 2)      # Output: 1

# Multiplication
print(3 * 2)      # Output: 6

# Division
print(3 / 2)      # Output: 1.5

# Exponential
print(3 ** 2)     # Output: 9

# Modulus (Remainder)
print(3 % 2)      # Output: 1

# Floor Division
print(3 // 2)     # Output: 1

üìä Data Types

Python supports several built-in data types to store different kinds of information:

Numbers

  • Integer: Whole numbers (positive, negative, or zero). Example: -3, -2, -1, 0, 1, 2, 3
  • Float: Decimal numbers. Example: -3.5, -2.25, -1.0, 0.0, 1.1, 2.2, 3.5
  • Complex: Numbers with imaginary parts. Example: 1+j, 2+4j

String

A collection of characters enclosed in single or double quotes:

'Shivamani'
'Python'
'I love teaching'
"30 Days of Python Challenge"

Boolean

A value that is either True or False (note the capitalization):

is_learning = True
is_tired = False

List

An ordered collection of items enclosed in square brackets. Lists can contain different data types:

[0, 1, 2, 3, 4, 5]  # List of integers
['Banana', 'Orange', 'Mango', 'Avocado']  # List of strings
['Banana', 10, False, 9.81]  # Mixed data types

Tuple

An ordered collection similar to lists, but immutable (cannot be modified after creation):

('Shiva', 'Sai', 'Aishu', 'Ravi')  # Names
('Earth', 'Jupiter', 'Neptune', 'Mars')  # Planets

Dictionary

An unordered collection of key-value pairs:

{
    'first_name': 'Shivamani',
    'last_name': 'Konam',
    'country': 'India',
    'age': 25,
    'is_married': False,
    'skills': ['Python', 'JavaScript', 'React']
}

Set

An unordered collection of unique items. Sets automatically remove duplicates:

{2, 4, 3, 5}
{3.14, 9.81, 2.7}  # Order doesn't matter in sets

Checking Data Types

Use the type() function to check the data type of any value:

print(type(10))          # 
print(type(3.14))        # 
print(type('Python'))    # 
print(type([1, 2, 3]))   # 
print(type((1, 2, 3)))   # 
print(type({'name': 'Python'}))  # 
print(type({1, 2, 3}))   # 
print(type(True))        #
üí° Tip: Understanding data types is fundamental. Each type has different properties and methods you can use.
