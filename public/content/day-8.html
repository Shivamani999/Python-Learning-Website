üìö What are Dictionaries?

A dictionary is a collection of unordered, modifiable (mutable) paired (key: value) data type. Unlike lists that use numeric indexes, dictionaries use keys to access values.

Creating a Dictionary

To create a dictionary we use curly brackets {} or the dict() built-in function. 

# Empty dictionary
empty_dict = {}

# Dictionary with data values
dct = {'key1':'value1', 'key2':'value2', 'key3':'value3'}

# Complex dictionary
person = {
    'first_name': 'Shiva',
    'last_name': 'Konam',
    'age': 250,
    'country': 'Finland',
    'is_married': True,
    'skills': ['JavaScript', 'React', 'Node', 'MongoDB', 'Python'],
    'address': {
        'street': 'Space street',
        'zipcode': '02210'
    }
}
üí° Note: Dictionary values can be any data type: strings, numbers, booleans, lists, tuples, sets, or even other dictionaries!
Dictionary Length

Use len() to check the number of key-value pairs in a dictionary. 

person = {
    'first_name': 'Shiva',
    'last_name': 'Konam',
    'age': 250,
    'country': 'Finland'
}
print(len(person))  # Output: 4
Accessing Dictionary Items

Access dictionary items by referring to the key name using square brackets or the get() method. 

person = {
    'first_name': 'Shiva',
    'country': 'Finland',
    'skills': ['JavaScript', 'Python']
}

# Using square brackets
print(person['first_name'])  # Shiva
print(person['skills'][0])   # JavaScript

# Using get() method (safer - returns None if key doesn't exist)
print(person.get('first_name'))  # Shiva
print(person.get('city'))        # None (no error)
Adding & Modifying Items

You can add new key-value pairs or modify existing ones by assigning a value to a key. 

person = {'first_name': 'Shiva', 'age': 250}

# Adding new items
person['job_title'] = 'Instructor'
person['city'] = 'Helsinki'

# Modifying existing items
person['age'] = 252

print(person)
# {'first_name': 'Shiva', 'age': 252, 'job_title': 'Instructor', 'city': 'Helsinki'}
Checking Keys

Use the in operator to check if a key exists in a dictionary. 

person = {'name': 'Shiva', 'age': 25}

print('name' in person)   # True
print('city' in person)   # False
Removing Items

Remove items using pop(), popitem(), or del. 

person = {'first_name': 'Shiva', 'last_name': 'Konam', 'age': 250}

# pop(key) - removes specific key
person.pop('age')

# popitem() - removes last inserted item
person.popitem()

# del - removes specific key
del person['first_name']
Dictionary Methods

Useful dictionary methods for working with keys, values, and items. 

person = {'name': 'Shiva', 'age': 25, 'city': 'Helsinki'}

# Get all keys
keys = person.keys()
print(keys)  # dict_keys(['name', 'age', 'city'])

# Get all values
values = person.values()
print(values)  # dict_values(['Shiva', 25, 'Helsinki'])

# Get all items as tuples
items = person.items()
print(items)  # dict_items([('name', 'Shiva'), ('age', 25), ('city', 'Helsinki')])

# Copy a dictionary
person_copy = person.copy()

# Clear all items
person.clear()
print(person)  # {}
